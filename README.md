# ComfyUI 插件拖拽冲突检查工具

这是一个用于检查 ComfyUI 插件冲突的小工具。

它可以扫描 ComfyUI 的 `custom_nodes` 插件文件夹，帮助找出哪些插件可能影响 PNG 图片或工作流 JSON 拖入 ComfyUI 后无反应的问题。

## 适合谁使用

适合不懂代码、但遇到下面问题的 ComfyUI 用户：

- 拖入 ComfyUI 生成的 PNG 图片后没有反应
- 拖入工作流 `.json` 文件后没有反应
- 不确定是哪个插件导致拖拽导入失效

## 工具特点

- 不修改任何 ComfyUI 文件
- 不删除、不移动任何插件
- 只读取插件代码并生成检查报告
- 可以导出 HTML 报告
- 适合普通用户排查问题

## 下载

请到右侧或下方的 Releases 页面下载最新版压缩包。

下载后解压，双击运行：

`ComfyUI插件拖拽冲突检查工具.exe`

## 使用方法

1. 打开工具
2. 选择你的 ComfyUI `custom_nodes` 插件文件夹
3. 点击“开始检查”
4. 查看结果
5. 如有需要，可导出 HTML 报告

## 风险说明

工具中的结果只是辅助判断：

- “很可能有问题”：建议优先排查
- “可能有影响”：表示插件碰到了相关功能入口，但不一定真的有问题
- “暂未发现问题”：暂时没有发现明显相关代码

最终是否真的是某个插件导致问题，建议关闭 ComfyUI 后临时移走该插件，再重启 ComfyUI 验证。
